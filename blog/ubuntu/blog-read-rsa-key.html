Read RSA Key In Ubuntu

Read private key normally

openssl rsa -noout -text -in key.private

Read public key normally

openssl rsa -noout -text -inform PEM -in key.pub -pubin

NOTE: To read the modulus in decimal, load it in heximal

openssl rsa -in key.pem -modulus -noout

Then copy the output in hex value to convert into decimal

echo '[hex value]' | bc
