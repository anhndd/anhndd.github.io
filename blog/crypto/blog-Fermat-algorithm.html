Fermat Algorithm

This page introduces the algorithm which helps find 2 prime number p and q by n

    
        t ← ⌈√n⌉
        z ← t2 - n
        When z is not a square:
            t ← t + 1
            z ← t2 - n
        return p = t + √z

For example, n = 51983

1. t = ⌈√n⌉ = 228

2. z = t2 - n = 1 (which is a square)

3. => p = t + √z = 228 + 1 = 229

4. => q = n/p = 227
