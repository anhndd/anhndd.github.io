Create Desktop Application In Ubuntu With .sh File

Step 1: Create file [appname].desktop at ~/.local/share/application

touch ~/.local/share/application/[nameapp].desktop

Step 2: modify .desktop file created

gedit ~/.local/share/application/[nameapp].desktop

Step 3: Add these code into file .desktop

[Desktop Entry]
Type=Application
Terminal=false
Name=[nameapp]
Icon=[path-of-icon-svg]/[namesvg].svg
Exec=[path-of-app-sh]/[namesh].sh
