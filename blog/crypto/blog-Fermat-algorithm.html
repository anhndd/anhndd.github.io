Fermat Algorithm

This page introduces the algorithm which helps find out 2 prime number p and q from n

	t ← ⌈√n⌉
	z ← t2 - n
	When z is not a square:
	    t ← t + 1
	    z ← t2 - n
	return p = t + √z

Example

With n = 51983

Step 1 Caculate t:

t = ⌈√n⌉ = 228

Step 2 Caculate z:

z = t2 - n = 1 (which is a square)

Step 3 Because z got a result as a square, break the loop and caculate p:

p = t + √z = 228 + 1 = 229

Step 4 Caculate q:

q = n/p = 227
