Change Default Termial To Terminator Right Click

Step 1: open file ~/.bashrc to modify

sudo gedit ~/.bashrc

Step 2: Add following code into ~/.bashrc

# change default terminal to terminator right click
if ps -o cmd= -p $(ps -o ppid= -p $$) | grep -q gnome; then
  nohup terminator &> /dev/null &
  sleep 0.1s
  exit
fi
